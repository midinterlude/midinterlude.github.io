# GitHub Pages Portfolio

A simple, dark-themed portfolio site built with HTML and CSS for GitHub Pages deployment.

## Features

- **Dark Theme**: Modern dark color scheme with subtle gradients
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Project Showcase**: Grid layout for displaying projects with hover effects
- **Biography Section**: About me section with skills display
- **Smooth Navigation**: Fixed header with smooth scrolling
- **Modern Typography**: Clean Inter font from Google Fonts

## Customization

### Personal Information
Replace the placeholder content in `index.html`:

1. **Name**: Change "Your Name" in the title, navigation, hero section, and footer
2. **Subtitle**: Update the hero subtitle with your actual title/role
3. **About Section**: Replace the placeholder text with your personal biography
4. **Skills**: Update the skill tags to match your actual technologies
5. **Projects**: Replace the sample projects with your real projects
6. **Contact**: Update email and social media links

### Styling
The CSS is fully customizable in `styles.css`:
- Colors are defined using hex values
- Responsive breakpoints at 768px and 480px
- Hover effects and transitions included
- Custom scrollbar styling

## Deployment to GitHub Pages

1. Push this repository to GitHub
2. Go to repository Settings > Pages
3. Select "Deploy from a branch" 
4. Choose the main branch and `/ (root)` folder
5. Your site will be available at `https://yourusername.github.io/repository-name`
